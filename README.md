# Iron Man: Stark Industries Bootcamp

## Backend (iron_man)

Backend from [AryanKhokale/iron_man](https://github.com/AryanKhokale/iron_man).

**Run:**
```bash
./run-backend.sh
# or
cd backend/iron_man && pip install -r requirements.txt && uvicorn main:app --reload
```

Backend runs at `http://127.0.0.1:8000`. Frontend `api-config.js` points here by default.

## Frontend

Open `index.html` in a browser or use a local server (e.g. Live Server).
